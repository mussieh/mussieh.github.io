Playfair Cipher Implementation

An implementation of the symmetric encryption scheme that uses substitution for encrypting text

Even though the Playfair cipher is old, this program is an important learning tool for understanding the weakness behind substitution ciphers

The Playfair cipher works on letter pairs (bigrams)

Hill Cipher Implementation

An implementation of the linear algebra-based cipher

The program allows for the encryption and decryption of text data based on a given key matrix

Even though this cipher is old as well, it is important to consider since it can be used to learn the problems associated with substitution ciphers regardless of the methods used (matrix algebra and modular arithmetic here)
